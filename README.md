# Walmart Sales Analytics – Power BI Dashboard

This project is an interactive **Walmart Sales Analytics** dashboard built in Power BI to analyze weekly store performance, seasonal patterns, and the impact of holidays on sales. It is designed as a portfolio project to demonstrate skills in data cleaning, DAX, and dashboard design. [web:7]

---

## Project Objectives

- Analyze total and average weekly sales across Walmart stores over time.
- Identify top-performing stores based on total sales.
- Compare sales performance between holiday and non‑holiday weeks.
- Provide an executive‑level view of YoY performance and key KPIs. [web:7]

## How to Open the Dashboard

1. Download the `Walmart_Sales_Analytics.pbix` file from this repository.
2. Open it using **Microsoft Power BI Desktop** (latest version recommended).
3. If prompted for data source paths, update them to your local file locations. [web:20]
